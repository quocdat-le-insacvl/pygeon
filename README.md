# Pygeon

## Attention:
- Ne push pas au branch Master svp

- Il vaut mieux d'utiliser : git clone , commit et tout, pour Merge facilement !

- Pour téléchargé:

  git clone https://github.com/quocdat-le-insacvl/pygeon.git
 
- Pour commit:

  git init
  
  git add .
  
  git commit -m "message"
  
- Pour push à github: (push to a new branch named : new_branch)

  git branch new_branch
  
  git push origin new_branch
  
- Pour faire un merge "pull request":
  
  Allez sur le new_branch (sur la site de github)
  
  Cliquez petit button : Pull request 
  
  Tapper le message => Create pull request
